# 🦋 Lorenz Attractor — 洛伦兹吸引子

> "一只蝴蝶扇动翅膀，可能引发一场龙卷风"
> 
> 混沌理论的iconic案例，最小代码量的数学奇观

## 📖 原理

洛伦兹吸引子由气象学家 **Edward Norton Lorenz** 于 1963 年在研究大气对流时偶然发现，是混沌理论的象征。

### 方程组

```
dx/dt = σ(y - x)
dy/dt = x(ρ - z) - y
dz/dt = xy - βz
```

### 参数

| 参数 | 值 | 含义 |
|------|-----|------|
| σ (Sigma) | 10 | Prandtl 数 |
| ρ (Rho) | 28 | Rayleigh 数 |
| β (Beta) | 8/3 | 几何因子 |

### 为什么是混沌？

- **对初值敏感**：两个相差 0.0001 的初始条件，经过一段时间后轨迹会完全分叉
- **确定性**：方程本身没有任何随机项
- **拓扑混合**：轨迹在吸引子上无限缠绕，永不重复
- **奇怪吸引子**：分数维几何体，维度约 2.06

## 🧬 蝴蝶效应演示

程序同时计算两条轨迹：
- **红色轨迹**：初始值 `(1.0, 1.0, 1.0)`
- **蓝色轨迹**：初始值 `(1.0001, 1.0, 1.0)`

仅 `0.01‰` 的差别，最终导致轨迹完全分离——这就是蝴蝶效应。

## 🎯 混沌哲学

> 混沌不是无序，而是 **确定性系统中的内在不可预测性**。
> 
> 洛伦兹吸引子告诉我们：即使我们可以精确写出自然法则，
> 测量的微小误差也会导致长期预测的彻底失败。
> 
> 这不是测量仪器的缺陷，而是**宇宙的本质**。
